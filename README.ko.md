# Daily Cat Wallpaper

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md) | 한국어

Daily Cat Wallpaper는 Rust와 Tauri 2로 만든 크로스 플랫폼 데스크톱 앱입니다. 단순히 고양이 사진을 주기적으로 바꾸는 앱이 아니라, 사용자 피드백을 학습하는 고양이 배경화면 시스템을 목표로 합니다. 고품질 고양이 이미지를 찾거나 생성하고, 실제 모니터 구성에 맞게 배경화면을 합성하며, 결과를 분석하고, 사용자의 평가를 다음 선택에 반영합니다.

Windows가 첫 번째 안정 대상 플랫폼입니다. macOS와 Linux는 정적 배경화면 설정을 지원하지만, 데스크톱 환경별 API 차이가 크기 때문에 현재는 Beta로 제공합니다.

## 제품 방향

현재 목표는 "Bing Wallpaper를 임의의 고양이 사진으로 바꾸는 것"이 아닙니다. 사용자의 의도에 더 잘 맞는 데스크톱 고양이 경험을 만드는 것입니다.

- 고양이는 사용자가 선택한 품종, 색상, 수량, 유형, 분위기, 장면에 최대한 맞아야 합니다.
- 2K 미만 이미지는 관리 갤러리에 들어가지 않아야 합니다.
- 투명 고양이 이미지는 고양이가 화면 위에 있는 것처럼 보이는 데스크톱 펫 스타일 배경화면에 사용할 수 있어야 합니다.
- 다중 모니터에서는 기본적으로 디스플레이 수에 맞춰 서로 다른 고양이를 사용합니다. 사용자가 1마리를 명시적으로 선택하면 여러 화면에서 같은 고양이를 재사용할 수 있습니다.
- 사용자는 다운로드되거나 생성된 이미지를 탐색하고, 마음에 드는 이미지를 직접 선택하며, 마음에 들지 않는 이미지를 삭제할 수 있어야 합니다.
- 앱은 폐쇄 루프를 가져야 합니다. 실제 데스크톱 결과를 분석하고, 좋아요/싫어요와 이유를 기록하며, 거부된 이미지가 다시 선택될 가능성을 낮춥니다.

## 주요 기능

- Rust 코어, Tauri 2 데스크톱 셸, React 설정 UI.
- UI와 README는 영어를 기본으로 하며, 간체 중국어, 번체 중국어, 일본어, 한국어를 지원합니다.
- Windows에서는 `SystemParametersInfoW`를 사용해 네이티브 정적 배경화면 설정을 수행하고, 매번 가상 데스크톱 전체 크기에 맞는 배경화면을 생성합니다.
- macOS는 AppleScript를 통한 배경화면 설정을 지원하며, 현재 Beta입니다.
- Linux는 GNOME, Cinnamon, Unity, KDE Plasma, Xfce를 best-effort 방식으로 지원하며, 현재 Beta입니다.
- TheCatAPI, Wikimedia Commons, 로컬 폴더, Pexels, Pixabay, Magnific, AI 생성 자산을 대상으로 품종과 선호도를 반영한 후보 계획을 수행합니다.
- 강제 HD 정책: `2560x1440` 미만 이미지는 관리 갤러리에 저장되기 전에 거부됩니다.
- 앱 데이터 디렉터리 아래에 고정 관리 갤러리를 두고, 다운로드 이미지와 AI 생성 이미지를 분리해 저장합니다.
- 내장 갤러리 브라우저에서 미리보기, 품질 배지, 수동 적용, 삭제를 지원합니다.
- AI 투명 고양이 PNG 생성으로 데스크톱 펫 스타일 cutout을 만들 수 있습니다. OpenAI는 기본적으로 투명 배경을 지원하는 이미지 모델을 사용하고, Google Nano Banana Pro는 사용자 API 키 기반 Beta 제공자입니다.
- 데스크톱 효과 분석은 최종 배경화면 크기, 가상 데스크톱 채움 여부, 해상도, placeholder 이미지, 투명 자산, 잘림 가능성, 선호도 일치 불확실성을 확인합니다.
- 피드백 학습은 좋아요/싫어요와 이유를 기록하고, 반복적으로 거부된 이미지를 이후 선택에서 피합니다.
- 호흡, 마우스 접근, 고양이 발 클릭, 키보드 Bongo, 소리 같은 상호작용 선호도를 설정할 수 있습니다. 소리는 기본적으로 꺼져 있습니다.

## 폐쇄 루프 학습

Daily Cat Wallpaper는 배경화면 선택을 지속적으로 개선되는 시스템으로 다룹니다.

1. 품종, 색상/유형, 고양이 수, 장면, 소스 우선순위, 갱신 주기, 상호작용, 언어를 읽습니다.
2. 로컬 파일, 관리 갤러리, 온라인 소스, AI 생성에서 후보 이미지를 찾습니다.
3. 저해상도 이미지는 저장 전에 거부합니다.
4. 실제 가상 데스크톱에 맞게 배경화면을 합성하고, 작업 표시줄과 아이콘 안전 영역도 고려합니다.
5. 최종 결과를 분석해 크기 부족, placeholder 이미지, 투명하지 않은 펫 자산, 잘림 가능성, 선호도 일치 불확실성을 보고합니다.
6. 사용자는 결과에 좋아요/싫어요를 남기고 이유를 입력할 수 있습니다.
7. 피드백을 앱 데이터 디렉터리에 저장하고, 거부된 이미지를 이후 선택에서 피합니다.

이 구조가 앱이 시간이 지날수록 사용자의 취향을 더 잘 이해하게 만드는 핵심입니다.

## 이미지 품질 정책

기본 최소 해상도는 `2560x1440`입니다. 저해상도 fallback은 비활성화되어 있습니다. 다운로드 이미지는 디스크에 저장되기 전에 검사되므로, 낮은 품질 후보가 관리 갤러리를 오염시키지 않습니다.

이미 존재하는 로컬 파일은 품질 배지와 함께 갤러리에 표시할 수 있어 사용자가 직접 정리할 수 있습니다.

## AI 고양이 이미지 생성

AI 생성 패널은 현재 UI 설정을 바탕으로 프롬프트를 만듭니다. 품종, 유형/색상, 장면, 수량, 투명 배경 선호도가 반영됩니다.

생성 이미지는 다음 위치에 저장됩니다.

- `cat-gallery/generated`

다운로드 이미지는 다음 위치에 저장됩니다.

- `cat-gallery/downloads`

투명 고양이 이미지의 목표는 배경이 없고 전신이 보이는 생동감 있는 PNG cutout입니다. 이 자산은 데스크톱 크기의 정적 배경화면에 합성됩니다. 진짜 always-on-top 투명 데스크톱 펫 레이어는 아직 향후 작업입니다. 현재 릴리스는 정적 배경화면 합성과 피드백 학습에 집중합니다.

## 설정

설정 UI에서 다루는 항목:

- 언어: 자동 감지 또는 명시적 선택.
- 고양이 품종과 유형/색상 선호도.
- 고양이 수 전략: 기본은 디스플레이 수에 맞춤, 또는 `1..5` 고정.
- 이미지 분위기와 장면.
- 이미지 소스 우선순위와 API 키.
- 로컬 갤러리 폴더.
- AI 생성 제공자, 모델, 장면 프롬프트, 수량, 자동 사용 여부.
- 갱신 일정: 로그인 시, 매일 지정 시간, 주기적 갱신, 수동 갱신.
- 플랫폼 모드와 시작 프로그램 설정.
- 상호작용 선호도와 소리 토글.

## 개발

필수 조건:

- `rustfmt`와 `clippy`를 포함한 Rust stable toolchain.
- Node.js 22 이상.
- 사용 중인 OS에 맞는 Tauri 플랫폼 의존성.

의존성 설치:

```powershell
npm install
```

검사 실행:

```powershell
cargo test --workspace
npm run build
cargo check
```

개발 모드 실행:

```powershell
npm run tauri dev
```

프로덕션 앱 빌드:

```powershell
npm run tauri:build
```

## 현재 제한

- Windows가 현재 주요 안정 대상입니다. macOS와 Linux 배경화면 설정은 Beta입니다.
- 투명 AI 고양이 이미지는 현재 정적 배경화면에 합성됩니다. 진짜 투명 오버레이 데스크톱 펫 레이어는 아직 완성되지 않았습니다.
- 온라인 이미지 품질은 제공자 API, rate limit, 사용자 API 키에 따라 달라집니다.
- 품종 매칭은 best-effort입니다. 사용자가 결과를 거부하면, 피드백 루프가 이후 선택의 우선 기준이 됩니다.
